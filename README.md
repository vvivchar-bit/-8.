## Мета роботи
Ознайомитися з поняттям патернів об’єктно-орієнтованого проєктування та вивчити особливості використання патерну **Singleton**, його класичної реалізації та реалізації Меєрса.

---

## Коротко про реалізації

### 1. Класичний Singleton
У класичному варіанті використовується:
- статичний вказівник на єдиний екземпляр класу;
- закритий конструктор;
- заборона копіювання об’єкта;
- статичний метод `getInstance()` для доступу до екземпляра.

### 2. Singleton Меєрса
У реалізації Меєрса:
- використовується локальний статичний об’єкт усередині методу `getInstance()`;
- не потрібно вручну видаляти об’єкт;
- реалізація є простішою та безпечнішою.
